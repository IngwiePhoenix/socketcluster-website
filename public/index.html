What is SocketCluster.io?

SocketCluster is a lightning fast, highly scalable HTTP + WebSocket (engine.io) server which lets you build multi-process real-time systems/apps that make use of ALL CPU cores on a machine/instance.

Removing the limitations of having to run your Node.js server as a single thread.

Technical Features

  • Sockets which are bound to the same browser (for example, across multiple tabs) share the same session.
  • Emit event notifications on any session to notify all sockets that belong to it.
  • The SocketCluster client (socketcluster-client) has an option to allow disconnected sockets to automatically (and seamlessly) reconnect if they lose the connection.
  • SocketCluster processes automagically respawn on crash - Worker and load balancer crashes are invisible to users.
  • It uses a memory store cluster called nData which you can use to store 'volatile' session data which relates to your sockets/sessions.
